initialize fields with a stationary atmosphere, see Benacchio (2014)

initializes fields for a simple wellbalancing testcase with a bathymetry bump and an initial flat sea surface and zero initial momentum. The general shape of the bump is given by BATHY_TYPE: 1: parabolically shaped bump with surrounding flat bathymetry 2: exponentially shaped bump Further parameters are given by ALPHA, BUMP_CENTRE, BUMP_HEIGHT. A small localized perturbation can be set with PERT_CENTRE and PERT_HEIGHT (note: in case PERT_HEIGHT > 0 only the initial condition is given by the exact solution). Note that the background fluid depth is set with BATHY_PARAM_DEPTH in the Parameters file.

initialize fields with a quasi-stationary vortex, see Vater (2013) Note that the background fluid depth is set with BATHY_PARAM_DEPTH in the Parameters file.

initializes fields with a quasi 1D tsunami runup onto a plane beach

initializes fields with Riemann problem, and computes exact solution

initializes fields for the oscillating lake test case from Thacker (1981). Parameters according to Gallardo et al. (2007) Note that h0, the center point fluid depth when surface is flat, is set with BATHY_PARAM_DEPTH in the Parameters file.

initializes fields with the Okushiri test case Note that the background fluid depth is set with BATHY_PARAM_DEPTH in the Parameters file.

initialize fields with topography from the Northsea

Initial condition for a linear standing wave, i.e. an initial saddle shaped fluid height with constant source_update and zero velocitites. The background depth is given by BATHY_PARAM_DEPTH in the Parameters file. This test is adapted from Iskandarani et al. (1995).

initialize fields with a Gaussian hump Note that the background fluid depth is set with BATHY_PARAM_DEPTH in the Parameters file.

initializes fields for long wave resonance in a cirular parabolic basin (Thacker 1981). Parameters according to Lynett et. al (2002) Note that h0, the center point fluid depth when surface is flat, is set with BATHY_PARAM_DEPTH in the Parameters file.

initialize fields using dambreak problem from Stelling & Duinmeijer: "A staggered conservative scheme for every froude number in rapidly varied shallow water flows", Int J Num Meth Fluids 43 (12), pp 1329â€“1354 (2003).

initialize velocity field with linear advection and SUD vortex

initialize fields for the conical island test case Note that the background fluid depth is set with BATHY_PARAM_DEPTH in the Parameters file.

initializes a channel with uniformly sloping source_update that contains a reservoir and a horizontal (total) water height.

Initial conditions for quasi 1D Tidal Basin with Bottom Convexity as in A. Balzano: "Evaluation of methods for numerical simulation of wetting and drying in shallow water flow models", Coastal Engineering 34, pp 83-107 (1998) Testcase 5.2
