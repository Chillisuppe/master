Detailed Description

reads bathymetry data from ETOPO data set

read bathymetry data from NetCDF file. This can be:

  • a GRD (NetCDF) file as being retrieved from http://www.marine-geo.org/portals/gmrt/
  • a NetCDF file in COARD convention (note: such a file must have a global attribute "Conventions" with value "COARDS")
